# Comparativo de Pastas

## Descrição
Script Python que compara o conteúdo de dois diretórios, identificando divergências entre arquivos com mesmo nome e gerando um relatório detalhado.

## Funcionalidades
- Compara arquivos entre duas pastas
- Filtra linhas baseado em assinaturas configuráveis
- Normaliza espaços em branco para evitar falsos positivos
- Gera relatório com linhas exclusivas de cada pasta
- Detecta arquivos ausentes na pasta B
